### [Nodejs](#nodejs)




















































Angular requires Node.js version 8.x or 10.x.

- To check your version, run 
```sh
node -v 
```
in a terminal/console window.

- To get Node.js, go to 
```sh
nodejs.org.
```

- Download and install the LTS version of node.
